# persuasion-principles
Most communication systems fail because they are designed for calm, rational users — and deployed in moments of cognitive stress.

## What this is
- A set of principles for designing communication systems
- Grounded in cognitive science, not copywriting
- Focused on moments of stress, confusion, and loss
- Supercharged by tech/AI

## What this is not
- Not copywriting advice
- Not growth optimization
- Not “empathy” as tone
- Not about maximizing compliance
